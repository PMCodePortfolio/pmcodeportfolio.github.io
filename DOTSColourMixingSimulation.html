DOTS Colour Mixing Simulation

To experiment with optimisation and multithreading techniques in Unity and C#, I created a sandbox project that incorporates the Data Oriented Technology Stack (DOTS), which involves the Entity Component System, Job System and Burst Compiler.

The project features a scene where hundreds of spherical entities of a certain colour are instanced. When they contact each other, they gradually exchange their colours to simulate mixing.

The simulation’s multithreaded systems utilise Entity Command Buffers to execute Structural Changes on pre-existing Sync Points and Unmanaged types are involved to support native compiling.

Despite the number of entities in the scene, the mentioned techniques allowed the simulation to run well within the approximate 4ms (for 240Hz displays) frame budget.

The VSync portion of the profiler graph demonstrates this efficiency further.
