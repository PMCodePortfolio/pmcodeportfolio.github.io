FPS Integration Tool

Under Grey Gear Systems, I worked on the FPS Integration Tool using Unity. The aim of this tool is to aid the development of First Person Shooter games by automatically handling the game mechanics and functionality.

For this project, I implemented:

  • Weapon firing, reloading, switching, activation…
  • Player & non-player movement
  • Universal health, damage & respawn systems
  • Adjustable UI including main, pause, success and fail menus
  • Player & non-player death phases
  • Collectable item system
  • Custom inspectors & ScriptableObjects

The code is purely written in C# and follows the SOLID programming principles while considering Abstraction and Reuse, with a substantial focus towards maintaining an effective system architecture to support future implementations.
