Inventory Web App

This item management application aims to track the possession of items along with their accompanying information and attributes, while involving an account feature to restrict such data to the authenticated user.

A collection of aspects that resemble this system are as follows:

Frontend - The client receives webpages with Razor-generated HTML, Bootstrap styling and JavaScript Fetch API.

Web Application - Developed with ASP.NET Core MVC with C#, a highly scalable, widely-used industry standard solution adopted to interact between the client and supportive services by separating the functionality into Model, DTO, View and Controller components.

SQL Container - Provides a relational database service for storing user account credentials for authentication and authorisation.

MongoDB Container - Holds a non-relational database to manage item records, where each can possess different fields due to the dynamics of their attribute rating collection.

Minio Container - Stores thumbnail images in a bucket accessible with record-specific keys. Such compartmentalisation from the database services mitigates a performance slowdown from large files.

Docker Compose - Controls how each container is configured and how they are deployed together.
